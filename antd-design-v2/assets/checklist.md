# 设计交付自查清单

交付设计稿前按此清单逐项检查，确保规范落地完整。

---

## Token 规范检查

- [ ] 所有色值使用 Token 名称标注，无直接写 `#1677FF` 等色值
- [ ] 正文 14px、辅助 12px、标题按层级 H1-H5 对应，无随意字号
- [ ] 间距值全部为 4px 的倍数（4/8/12/16/20/24/32/40/48/64）
- [ ] 圆角：组件级 6px，卡片 8px，小组件 4px，与 Token 一致
- [ ] 功能色使用语义色：成功绿 `colorSuccess`、错误红 `colorError`、警告橙 `colorWarning`
- [ ] 文本颜色：标题 `.88` 透明度、正文 `.88`、次要 `.65`、禁用 `.25`

---

## 组件状态完整性检查

**通用状态（所有可交互组件必须覆盖）：**
- [ ] 默认态（Default）
- [ ] 悬停态（Hover）
- [ ] 激活/选中态（Active / Selected）
- [ ] 禁用态（Disabled）
- [ ] 加载态（Loading，异步操作组件）
- [ ] 错误态（Error，表单校验）

**表单组件额外检查：**
- [ ] 必填字段有 `*` 标记
- [ ] 所有字段有 placeholder 说明
- [ ] 校验错误信息已设计，位置在字段下方
- [ ] 字段禁用态背景色 `colorFillQuaternary`，文字 `colorTextDisabled`

**表格额外检查：**
- [ ] 空数据态已设计（有引导文字和操作入口）
- [ ] 加载态已设计（Spin 或 Skeleton）
- [ ] 操作列已固定 `fixed="right"`
- [ ] 长文本列有截断 + Tooltip 方案
- [ ] 状态列 Tag 使用语义色

**按钮额外检查：**
- [ ] 主操作按钮（Primary）每个区域只有一个
- [ ] 危险操作使用 `danger` 类型
- [ ] 危险操作有 Popconfirm 或 Modal.confirm 二次确认

---

## 反馈闭环检查

- [ ] 所有提交/保存操作有 loading 态
- [ ] 操作成功有 message.success 提示（2-3s）
- [ ] 操作失败有 message.error 提示，含失败原因
- [ ] 删除/禁用等不可逆操作有 Popconfirm 二次确认
- [ ] 系统级通知使用 Notification（右上角）
- [ ] 空态有具体说明和操作引导，无"暂无数据"四字了事

---

## 页面结构检查

- [ ] 面包屑路径正确，可点击父级节点
- [ ] 页面标题与导航菜单名称一致
- [ ] 操作区按钮顺序：主操作在右/上方，次要操作在左/下方
- [ ] 搜索条件超 4 个有折叠方案
- [ ] 长表单按信息归属分组（Card + Divider）
- [ ] 底部 / 侧边操作栏有 `position: sticky` 固定

---

## 布局技术规范检查（交付给前端时）

- [ ] 最外层 Layout 使用 `height: 100vh; overflow: hidden`
- [ ] 内容滚动区使用 `overflow-y: auto`，不在 body / html 层滚动
- [ ] flex 子项包含 `min-height: 0` 防止溢出
- [ ] 图标尺寸明确：导航 16px，按钮内 14-16px，空态插图 64-96px
- [ ] 固定宽度列数值已标注，无"自适应"不明确描述

---

## 标注规范检查

- [ ] 组件状态全部覆盖，非仅默认态一种
- [ ] 色彩标注使用 Token 名称
- [ ] 间距使用具体数值（如 `16px`），不写"适当间距"
- [ ] 交互说明：点击触发什么、出现什么、如何关闭
- [ ] 响应式断点已说明（如适用）
- [ ] 与前端对齐：哪些字段是动态数据、字段长度边界情况

---

## 设计资产交付检查

- [ ] 切图命名规范，与前端约定一致
- [ ] 导出尺寸：1x/2x/3x（如有图片资源）
- [ ] Figma 图层结构清晰，组件有正确命名
- [ ] 原型流程图已链接（如有交互演示需求）
- [ ] 设计稿已归档到项目对应文件夹，权限已开放给开发

---

## 最终上线前检查

- [ ] 走查真实数据：长文本是否截断正常、数字是否对齐
- [ ] 在 1280px / 1440px / 1920px 三个宽度验证布局
- [ ] 深色模式（如系统支持）颜色变量是否自动适配
- [ ] 无障碍：操作按钮有 aria-label，颜色对比度达标（WCAG AA）
